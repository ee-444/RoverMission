void HMC6352compass::enterCalibration ( )

HMC6352compass method for calibration mode.

Enter user calibration mode. 2 complete rotations should occur (720 degrees) in 20 seconds. The LED will flash a rate of 1hz during this process. The rotation should be smooth and uniform
